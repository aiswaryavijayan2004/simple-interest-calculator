# Code of Conduct

## Our Pledge
We pledge to make participation in our project a harassment-free experience for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment include:
- Using welcoming and inclusive language
- Being respectful of differing viewpoints
- Accepting constructive criticism gracefully

## Enforcement Responsibilities
Project maintainers are responsible for clarifying standards of acceptable behavior.

## Scope
This Code of Conduct applies within all project spaces.

## Enforcement
Instances of abusive behavior may be reported to the project team.
